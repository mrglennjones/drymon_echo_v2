## ⚡ INSTANT START

### 1️⃣ Load the Effect

- Connect your **NTS-3**

- Done!

| Control | What It Does | Start Here |
|---------|--------------|------------|

## 🎮 TOUCHPAD TRICKS

### Default Mode (TOUCH: Off)
- **X-axis** = Lag Time (sweep from flange to echo)
- **Y-axis** = Wobble (add movement)

### Performance Mode (TOUCH: On)
- **X-axis** = Blend (dry → wet control)
- **Y-axis** = Wobble (add movement)

**💡 Tip**: Turn TOUCH to **On** for DJ-style wet/dry sweeps!

---

## 🎯 WHAT DOES WHAT?

### LAG (Lag Time) - The Most Important Parameter

| Range | Effect Type | Delay Time | Best For |
|-------|-------------|------------|----------|
| **0-10%** | **Flange** | -0.3ms to 3ms | Swooshing, thickness |
| **10-30%** | **Chorus** | 3ms to 50ms | Doubling, width |
| **30-60%** | **Slapback** | 50ms to 150ms | Rockabilly, vocals |
| **60-100%** | **Echo** | 150ms to 500ms | Ambient, delays |

---

### TYPE - Three Different Modes

**Sum/Classic** (Default)  
✅ Wide stereo doubling  
✅ Smooth tape saturation  
✅ Most versatile

**Invert/Classic**  
✅ Through-zero flanging  
✅ Phase cancellation  
✅ Dramatic sweeps

**Bounce/Cassette**  
✅ Ping-pong stereo  
✅ Lo-fi cassette compression  
✅ Wide stereo delays

## ⚠️ COMMON QUESTIONS

**Q: Why does it sound different on different instruments?**  
A: Tape saturation responds to input level (like real tape!)

**Q: The touchpad isn't working!**  
A: Check if TOUCH is set to On (for custom X/Y control)

**Q: Can I use this on everything?**  
A: Yes! Try low SAT (10-20%) on the master bus for subtle warmth
